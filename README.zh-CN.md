# 🧠 AskPeer — 学术专家按需网络

> **研究卡住了？找个做过的人问一下。**

AskPeer 把研究者与领域专家连接起来，做按需咨询。不是审稿。不是合作谈判。就是：你有问题 → 我们找到知道答案的人 → 你们聊。

---

## 为什么

学术知识交换有三件事烂掉了：

| 痛点 | 现状 | AskPeer |
|------|------|---------|
| **审稿** | 3-6个月，无偿，质量随机 | 不归我们管。我们做咨询。 |
| **方法学求助** | 冷邮件给论文作者，祈祷回复 | AI 几小时内匹配对的人 |
| **跨领域专家** | 靠老板的人脉。就这。 | 全球专家图谱，无看门人 |

a16z 投了 $22.75M 的 [Ethos](https://askethos.com) 在企业端跑通了这个模式（$500/hr 专家通话）。AskPeer 把它搬进学术界——用学术合适的激励方式，不只有钱。

---

## AskPeer 是什么

```
研究者有问题
      │
      ▼
  AI 分析问题
  + 搜索专家图谱
      │
      ▼
  Top 3 匹配专家
      │
      ▼
  研究者选一个
      │
      ▼
  30分钟咨询
  (通话 / 异步 / 线下)
      │
      ▼
  问题解决。评价 & 感谢。
```

**交付方式** — 灵活，不只是付费通话：
- 🎥 **实时通话** — 30min Zoom / 视频
- ✍️ **异步反馈** — 48h 内书面回复
- 🤝 **合作** — 双方愿意就转共同作者

**激励方式** — 符合学术场景：
- 专家自定费率（或免费）
- 机构认可 / 院系荣誉
- 合作署名可选（咨询产生实质贡献时）

## MVP — 第一阶段

极简匹配引擎。无支付。无登录墙。

```
┌─────────────────────────────────────────┐
│  AskPeer MVP                            │
│                                         │
│  [ 描述你的研究问题...                 ] │
│  [ 你的领域: _______ ]                 │
│  [ 偏好方式: 通话 / 异步 / 随意 ]      │
│                                         │
│         [ 找专家 ]                      │
│                                         │
│  ── 匹配结果 ────────────────────────   │
│  张教授 — 冷冻电镜，膜蛋白              │
│  李博士 — 分子动力学，力场              │
│  王老师 — 统计分析，R                   │
│                                         │
│  [ 联系张教授 ]                         │
└─────────────────────────────────────────┘
```

MVP 做的事：**问题 → 匹配 → 介绍邮件。** 无平台通话，无支付——只负责连接两个研究者。

## 技术方案

| 层 | 技术 |
|----|------|
| 专家信息提取 | LLM 读论文/CV → 结构化专家图谱 |
| 问题理解 | LLM 分类领域 + 方法 + 深度 |
| 匹配 | 专家嵌入向量的相似度搜索 |
| 前端 | 极简 Web UI（Streamlit 或纯 HTML） |
| 后端 | Python + SQLite（MVP） |
